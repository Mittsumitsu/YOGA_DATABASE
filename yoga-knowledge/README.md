# Yoga Knowledge Database

個人用ヨガ知識ベース。原典に基づく正確な情報のみを収録。
ExBrain互換設計 / GitHub管理 / Claudeプロジェクト連携。

## 収録ソース（全てPublic Domain）
- パタンジャリ『ヨーガ・スートラ』全196スートラ（Johnston訳 1912）
- 『ハタヨガ・プラディーピカー』（15世紀・Svātmārāma著）
- 『バガヴァッド・ギーター』（関連章のみ）

## フォルダ構成
| フォルダ | 内容 |
|---|---|
| 01_Vocabulary | サンスクリット用語集（カタカナ読み付き）|
| 02_Sutras | スートラ原典（章・節番号で引ける）|
| 03_Philosophy | 哲学体系（サーンキャ・ヴェーダーンタ等）|
| 04_Asana | アーサナ名の語源・意味 |
| 05_Sources | 原典テキスト情報 |
