# スヴァスティカーサナ（svastikāsana）

## 区分
**区分A — HYP確認済み**（HYP 第1章「Swastika-āsana」）

## 読み
スヴァスティカーサナ

## 語源
svastika（スヴァスティカ / 吉祥の標識・卍）＋ āsana（アーサナ / 座法）。
sva（自己）＋ asti（あり・善し）の合成語で「善き兆し」を意味する。
直訳は「吉祥の座法」。

## 原典記述（HYP 第1章）

> Having kept both the hands under both the thighs, with the body straight, when one sits calmly in this posture, it is called Swastika.

*出典: 05_Sources/HYP_Svatmarama_fulltext.md — Chapter 1, Swastika-āsana*

## 解説
両手を両腿の下に置き、身体をまっすぐにして静かに座る姿勢。
HYPでは「I am going to describe certain āsanas which have been adopted by Munis like Vasistha, etc., and Yogis and Yoginis like Matsyendra, etc.」とした上で最初に挙げられるアーサナのひとつ。
