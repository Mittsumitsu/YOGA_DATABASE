# アシュタンガ（Aṣṭāṅga）

## 哲学体系
パタンジャリ・ヨーガ

## 核心的な意味
Aṣṭāṅga は aṣṭa（八）＋ aṅga（支・部分）から成る語で、「八つの支分」を意味する。ヨーガ・スートラでは、実践体系として八支則が示される。

八支則は次の八つ。

1. ヤマ（yama / 制戒）
2. ニヤマ（niyama / 勧戒）
3. アーサナ（āsana / 坐法・姿勢）
4. プラーナーヤーマ（prāṇāyāma / 呼吸の制御）
5. プラティヤーハーラ（pratyāhāra / 感覚の制御）
6. ダーラナー（dhāraṇā / 集中）
7. ディヤーナ（dhyāna / 瞑想）
8. サマーディ（samādhi / 三昧）

## 出典
> ヨーガ・スートラ 第2章29節
> Sanskrit: yama-niyama-āsana-prāṇāyāma-pratyāhāra-dhāraṇā-dhyāna-samādhayo 'ṣṭāv aṅgāni

Johnston訳（1912）では、八つの手段として制戒・勧戒・姿勢・呼吸制御・感覚の撤収・注意・瞑想・観照が列挙される。
